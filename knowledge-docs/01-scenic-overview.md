# 云隐山风景区概览

## 景区简介
云隐山风景区位于中国东南部，总面积约120平方公里，是国家5A级旅游景区、世界地质公园。景区以其险峻的山峰、幽深的峡谷、悠久的历史文化和丰富的生物多样性而闻名。主峰望岳峰海拔1080米，常年云雾缭绕，故名"云隐"。

## 地理位置与交通
云隐山距最近的高铁站约30公里，景区设有三个游客中心：南门游客中心（主入口）、北门游客中心、西门游客中心。景区内部交通以环保观光车为主，共设12个停靠站点，覆盖所有主要景点。建议游览时间为1-2天。

## 门票信息
- 成人票：120元/人
- 学生票/老人票：60元/人（需提供有效证件）
- 儿童票：1.2米以下免票，1.2-1.5米半票
- 景区开放时间：旺季（4月-10月）6:30-18:00，淡季（11月-3月）7:00-17:30
- 观光车通票：50元/人（当日有效，不限次数乘坐）

## 主要景点
1. **望岳峰**：海拔1080米，景区最高点，以云海日出闻名
2. **龙隐寺**：始建于唐代的古刹，国家重点文物保护单位
3. **翠微湖**：高山堰塞湖，面积约2平方公里，可乘船游览
4. **飞瀑峡**：全长3公里的峡谷，有大小瀑布12处
5. **古驿道**：明清时期的茶马古道遗址，全长约5公里

## 游览路线推荐
- **经典一日游**：南门进 → 观光车至望岳峰索道站 → 登顶望岳峰 → 龙隐寺 → 翠微湖午餐 → 飞瀑峡 → 南门出
- **深度两日游**：第一天南门进游览核心景点，夜宿景区内云隐山庄；第二天走古驿道，探索峡谷深处

## 服务设施
景区内设有游客服务中心、医疗急救站、母婴室、行李寄存处等配套设施。餐饮方面有多家特色餐厅，提供地方美食和简餐。住宿可选择景区内的云隐山庄（四星级）或周边的民宿客栈。
